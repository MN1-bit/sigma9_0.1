## 발견된 이슈

### 1. ❌ 타임프레임 변경 실패

**현상**: 1m, 5m, 15m, 1h, 1D 버튼 클릭 시 차트가 갱신되지 않음

**예상 원인**: (분석 필요)

---

### 2. ❌ 차트 색상 변경 불가

**현상**: 캔들 색상, 볼륨 색상 등이 하드코딩되어 있어 설정에서 변경 불가

**요구사항**:
- `settings.yaml`에 차트 색상 설정 추가
- 상승/하락 캔들 색상
- 볼륨 바 색상
- 배경/그리드 색상

---

### 3. ❌ Crosshair 표시 설정 불가

**현상**: Crosshair의 가격/시간 라벨 색상, 폰트가 고정되어 있음

**요구사항**:
- `settings.yaml`에서 crosshair 스타일 조절 가능
- 라벨 배경색
- 폰트 크기/색상
