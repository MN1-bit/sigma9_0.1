## Part 1: 핵심 철학 비교 토론

### 라운드 1: "엣지의 본질은 무엇인가?"

**Claude 전략서**: 엣지는 **4가지 비대칭**에서 나온다 — 정보, 실행, 심리, 리스크. Float Rotation Theory와 Liquidity Vacuum 개념을 통해 "참가자 피로도"를 읽는다.

**ChatGPT 전략서**: 엣지는 **예측이 아니라 필터링**이다. 나쁜 기대값을 제거하고, 실행 가능성(유동성/스프레드)을 기반으로 비용 이후 EV를 확보하는 것.

**Gemini 전략서**: 엣지는 **공매도 사냥(Hunt the Shorts)**이다. 갇힌 숏 세력의 강제 커버링 순간을 Level 2로 읽어내는 것이 핵심.

**Perplexity 전략서**: 엣지는 **리스크 구조와 유동성 구조를 남들보다 먼저 보는 능력**이다. 승률보다 R-multiple과 기대값 최적화에 집중.

**토론 결론**: ChatGPT와 Perplexity가 가장 실전적인 관점을 제시한다. "예측"보다 "필터링과 구조 파악"이라는 공통점이 있으며, 이것이 지속 가능한 엣지의 본질에 가깝다.

## Part 2: 플레이북 패턴 비교

### 공통 패턴 분석

| 패턴 | Claude | ChatGPT | Gemini | Perplexity |
|------|--------|---------|--------|------------|
| VWAP Reclaim | ✅ 상세 | ✅ 상세 | ✅ 상세 | ✅ (구조 트리거로 포함) |
| HOD Break | ✅ 흡수 필수 | ✅ 3요소 동시충족 | 명시 없음 | ✅ HOD retest |
| Gap and Go | 언급만 | ✅ 프리마켓 하이/ORB | ✅ 상세 | ✅ 프론트사이드 |
| 홀트 플레이 | 위험 경고만 | ✅ 상세 | ✅ LULD 상세 | 언급 없음 |
| FRD (First Red Day) | ✅ 상세 | 없음 | 없음 | 없음 |
| 워시아웃 리버설 | 없음 | ✅ 상세 | 없음 | ✅ 피크 플레이 |
| Short Squeeze | 언급만 | 없음 | ✅ 상세 | 없음 |

**토론 결론**: 
- **VWAP 플레이**: 4개 전략 모두 핵심으로 채택 → 필수 포함
- **HOD Break**: ChatGPT의 "3요소 동시충족(압축+유동성+체결가속)" 조건이 가장 엄격하고 안전
- **홀트 플레이**: ChatGPT와 Gemini가 상세하게 다룸, 고위험-고수익 패턴
- **FRD Bounce**: Claude만 언급, 멀티데이 러너에서 유효한 패턴

### 금지 조건 비교 (가장 중요)

**Claude의 절대 금지**:
- 물타기, 추격 매수, 역추세 매매, 홀트 중 무계획 홀딩, 오버사이징

**ChatGPT의 거래 금지 신호**:
1. 스프레드 체계적 확대
2. 체결은 붙는데 가격 못 감 (흡수/분배)
3. 재돌파 실패 반복
4. 유동성 급감
5. 손절이 지켜지지 않을 것 같은 구조

**Gemini의 절대 금지**:
- 물타기, 오퍼링 확인 없이 홀딩, FOMO 매수
- **오버나잇 완전 금지** (가장 강경)

**Perplexity의 금지**:
- 백사이드에서 역추세 매수
- 손절 라인 불명확한 자리 진입
- 연속 3회 손절 시 해당 패턴 당일 중지

**토론 결론**: ChatGPT의 5가지 거래 금지 신호가 가장 실용적이고 체계적. Perplexity의 "연속 3회 손절 시 패턴 중지" 규칙은 심리 관리 측면에서 우수. Gemini의 "오버나잇 완전 금지"는 마이크로캡에서 필수.

## Part 3: 매매 운용 비교

### 포지션 사이징

| 전략서 | 방식 |
|--------|------|
| Claude | ATR 기반 동적 + 달러 고정 ($100-500) |
| ChatGPT | 계획된 R 기준, 유동성에 따라 가변 |
| Gemini | 계좌의 1-2%, 기술적 손절폭에 맞춘 수량 |
| Perplexity | 0.5-1R 고정, 유동성 얇은 곳에서 축소 |

**토론 결론**: Perplexity와 ChatGPT의 "R 기반 사이징"이 가장 체계적. 슬리피지/유동성에 따른 가변 조절은 필수.

### 스케일링 전략

**Claude**: 진입 33/33/34, 청산 50/25/25
**ChatGPT**: 패턴별 상이
**Gemini**: 확신 높으면 1회, 애매하면 파일럿 → 불타기
**Perplexity**: 1차 30-50% → 확인 후 2차 추가, 손절 1R 내 유지

**토론 결론**: Perplexity의 스케일링이 가장 상세하고 리스크 관리가 철저. "추가 진입 시에도 전체 R을 1 이내로 유지"하는 원칙이 핵심.

## Part 5: 최적 하이브리드 구성

ChatGPT를 기본 프레임워크로 하되, 다른 전략에서 차용할 요소:

### From Claude:
- **Float Rotation Theory**: 플로트 5회 이상 회전 시 참가자 피로도 → 거래 금지 신호로 추가
- **Tier별 데이터 분류**: 필수/엣지강화/고급 분석 3단계 체계
- **FRD Bounce 패턴**: 멀티데이  러너 전용 플레이북으로 추가

### From Gemini:
- **종목 필터 수치**: Float<10M, RVOL>5x, Gap>20% (더 엄격한 기준)
- **오버나잇 완전 금지**: 마이크로캡에서 절대 규칙화
- **LULD Halt 플레이**: 고급 패턴으로 추가
- **Dilution Checker**: SEC EDGAR 크롤링으로 ATM 여부 자동 체크
- **Float Rotation 실시간 계산**: 시스템 필수 기능으로 포함

### From Perplexity:
- **가격 필터**: $2-10 구간 (슬리피지 관리)
- **R-multiple 기반 사이징**: 0.5-1R 고정, 유동성에 따라 가변
- **스케일링 상세 규칙**: 추가 진입 시에도 총 R을 1 이내 유지
- **연속 3회 손절 시 패턴 중지**: 심리 관리 규칙
- **데일리 드로다운 한도**: -3R 또는 -3% 초과 시 당일 거래 중단

- **오버나잇**: **절대 금지**

1. **VWAP Reclaim** (4개 전략 공통)
2. **HOD Break with 3요소** (ChatGPT 기준: 압축+유동성+체결가속)
3. **First Pullback / Bull Flag**
4. **Gap and Go / ORB**
5. **홀트 재개** (고급)
6. **FRD Bounce** (멀티데이용)

### 4. 거래 금지 신호 (ChatGPT + 보강)
1. 스프레드 체계적 확대
2. 체결 붙는데 가격 못 감
3. 재돌파 실패 반복
4. 유동성 급감
5. Float rotation 5회 이상 (Claude 추가)
6. 손절 지켜지지 않을 것 같은 구조

### 5. 리스크 관리 (Perplexity 기준 채택)
- 트레이드당: 0.5-1R
- 스케일 인: 총 R이 1 넘지 않도록
- 연속 3회 손절: 해당 패턴 당일 중지
- 데일리 드로다운: -3R 초과 시 거래 중단
- 오버나잇: 금지

### 6. 시스템 필수 기능
1. Float Rotation 실시간 계산
2. Dilution Checker (SEC EDGAR 연동)
3. VWAP/RVOL 로컬 계산 (속도 확보)
4. 거래 금지 신호 자동 알림
