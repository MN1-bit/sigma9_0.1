# .env.example

## 기본 정보
| 항목 | 값 |
|------|---|
| **경로** | `.env.example` |
| **역할** | 환경 변수 템플릿 (실제 .env 파일 생성 가이드) |
| **라인 수** | 27 |

## 환경 변수

### IBKR 연결
| 변수 | 기본값 | 설명 |
|------|--------|------|
| `IB_HOST` | `127.0.0.1` | IB Gateway 호스트 |
| `IB_PORT` | `4002` | Paper: 4002, Live: 4001 |
| `IB_CLIENT_ID` | `1` | 고유 클라이언트 ID |
| `IB_ACCOUNT` | (빈값) | 계좌 ID (자동 감지) |

### API 키
| 변수 | 설명 |
|------|------|
| `POLYGON_API_KEY` | Polygon.io API 키 |
| `OPENAI_API_KEY` | OpenAI API 키 (선택) |
| `ANTHROPIC_API_KEY` | Anthropic API 키 (선택) |

## 사용법
```bash
cp .env.example .env
# .env 파일 수정
```

> ⚠️ `.env` 파일은 절대 Git에 커밋하지 마세요!
