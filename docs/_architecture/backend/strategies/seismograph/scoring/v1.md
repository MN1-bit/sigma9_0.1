| 우선순위 | 점수 | 조건 | 의미 |
|---------|------|------|------|
| 1순위 | 100점 | Tight Range + OBV | 🔥 폭발 임박 |
| 2순위 | 80점 | Tight Range only | 높은 관심 |
| 3순위 | 70점 | Accumulation + OBV | 관심 대상 |
| 4순위 | 50점 | Accumulation only | 추적 중 |
| 5순위 | 30점 | OBV only | 모니터링 |
| 6순위 | 10점 | Volume Dryout only | 관찰 대상 |
