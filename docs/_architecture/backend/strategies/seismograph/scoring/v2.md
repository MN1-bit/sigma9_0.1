## 함수

### `calculate_score_v2(daily_data, intensities, weights?) -> float`
> 가중합 연속 점수 (V1의 Step 함수 → 연속 함수)

#### 수식
```
Score = 100 × Σ(w_i × I_i)
```
- `w_i`: 신호 가중치
- `I_i`: 신호 강도 (0.0~1.0)

#### Args
- `daily_data`: 일봉 데이터
- `intensities`: 시그널 강도 dict (예: `{"tight_range": 0.8, ...}`)
- `weights`: 가중치 (기본: `SCORE_WEIGHTS`)

#### Returns
- `float`: 0.0~100.0 연속 점수
